# Japa — Project Documentation

GitHub is the source of truth for this project documentation. Notion indexes this file in the Priyansh App Factory Command Center.

## 00. Executive Summary
Japa is a local-first mantra repetition app. Users choose or create a mantra, set a target count, complete repetitions through a digital mala interaction, and track practice history. It is for users who want a quiet, respectful, distraction-free practice tool. The end product should include mantra selection, count targets, bead/count UI, session completion, history, reminders, and local-first storage.

## 01. Product
MVP scope: mantra selection/create, count target, digital mala UI, session completion, history/streak, optional reminder.

## 02. Design
Quiet, bead-like, tactile, devotional. Screens: Home, Mantra Select, Practice, Completion, History, Settings.

## 03. Frontend Technical
SwiftUI local-first app. Store mantras, sessions, streaks, and preferences locally. Optional reminders.

## 04. Backend Technical
No backend for v1. Future services may include content catalog, optional sync, or audio downloads.

## 05. Business
Business model: free core, optional audio/content packs, supporter unlock, or bundle with a larger spiritual product.

## 06. Marketing
Positioning: a quiet digital mala for daily japa. Channels: community groups, practice demos, festival content.

## 07. User Acquisition
Beta with practice users and community testers. Metrics: first session, target completion, repeat practice, reminder opt-in, retention.

## 08. Execution
Plan: audit repo, resolve overlap, freeze count model, build practice UI, add history/reminders, QA/content review.

## 09. QA
Test count increments, accidental taps, pause/resume, completion, history, reminders, reset, accessibility, and content review.

## 10. Legal / Compliance
Document local data handling and optional reminders. Keep content respectful and reviewed.

## 11. Operations
Release process: internal practice test, content review, community beta, TestFlight. Post-launch: audio, widgets, Digital Temple integration.
